Description
What the dickens is an anagram game. All words come from novels by Charles Dickens. The game is simple, just solve as many anagrams as you can in three minutes. Title
msgpack visualizer
Screenshots
Description
The msgpack visualizer is a tool to help debug problems with messages that have been encoded using the msgpack serialisation protocol. It decodes a base64 string and presents it as a JSON object. As well as presenting it as a JSON object it also prints out which bytes correspond to which data type and value. It is written in Javascript so that it can be easily used from a web browser.
